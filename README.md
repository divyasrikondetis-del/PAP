# 🌿 PAP - Precision Agriculture Platform

AI-powered farming assistant — detect crop diseases, check market prices, rent equipment & more.

## Features

- 🌾 **Crop Disease Detection** - AI-powered scanner to identify crop diseases
- 📊 **Market Prices** - Real-time crop market prices
- 🚜 **Equipment Rental** - Rent farming equipment easily
- 📱 **Mobile Friendly** - Works on all devices
- 🌍 **Multi-language Support** - English, தமிழ், and more

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/divyasrikondetis-del/PAP.git

# Navigate to project
cd PAP

# Install dependencies
npm install

# Run development server
npm run dev
